My Own DBMS - Development Progress

Project Overview

My Own DBMS is a free and open-source database management system built using Go. The main objective is to develop a lightweight and efficient DBMS for learning purposes, focusing on SQL parsing, storage management, and query execution.

Current Progress

So far, the project includes essential structures to create table files and parse SQL commands. The following components are already implemented:

  • SQL Sanitizer: Cleans and validates SQL input.
  • Tokenizer: Breaks down SQL statements into meaningful tokens.
  • Lexical Analyzer: Identifies keywords, identifiers, and syntax elements.

The next major component under development is the SQL Runner, which will execute queries.

Next Steps

The upcoming milestones for the project include:

  • Implementing data storage and retrieval for tables.
  • Developing query execution logic.
  • Enhancing the database file structure for performance optimization.
